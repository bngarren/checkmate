## ☑️ Installation

- install using [lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{
    "bngarren/checkmate.nvim",
    opts = {
        -- your configuration here
        -- or leave empty to use defaults
        todo_markers = {
            unchecked = "□",
            checked = "✔",
  },

    }
}
```
